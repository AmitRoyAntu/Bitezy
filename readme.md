# Bitezy

Bitezy is a campus-focused food marketplace that connects canteens, food carts, and cafeterias with students, faculty, and other buyers in a unified platform. Developed as part of an Internet Programming project, it is a multi-role web application with dedicated experiences for customers, sellers, and admins. Its goal is to make campus food ordering faster, more convenient, and well-organized for everyone involved.

## Quick Start

1. Install dependencies from the project root:
```
npm install
```

2. Create your environment file:
- At `backend/.env`
- Copy values from `backend/example.env`
- Update values if needed

3. Start frontend + backend together:
```
npm run dev
```

4. Open the app:
- Frontend: `http://localhost:3002`
- Backend API: `http://localhost:8002`

## What You Need
- Node.js
- npm
- MongoDB (local or cloud URI)
